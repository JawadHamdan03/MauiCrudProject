# MAUI User Management App

A simple .NET MAUI application that demonstrates CRUD (Create, Read, Update, Delete) operations on User entities using Entity Framework Core and SQLite.

## Features

- Create new users
- View all users
- Update existing user information
- Delete users
- Local SQLite database storage
- Cross-platform support (Android, iOS, Windows, macOS)

## Technologies Used

- .NET MAUI
- Entity Framework Core
- SQLite

## Prerequisites

- .NET 10 SDK 
- Visual Studio 2026 (v17.3 or later) with .NET MAUI workload installed
- OR Visual Studio Code with .NET MAUI extension


## Database

The application uses SQLite as the local database. The database file is automatically created on first run and stored in the app's local data directory.

## Usage

The app provides a simple interface to:
- Add new users with their details
- Browse the list of existing users
- Edit user information
- Remove users from the database
